# Excel Sales Analysis Portfolio Project 

## Project Overview 
This project analyzes a dataset using Microsoft Excel to summarize the sales data.

## Dataset 
The dataset contains: Pivot Table and Power Pivot.

## Tools Used
- Microsoft Excel
- Excel formulas
- Summary tables
- PivotTable-style analysis
- Excel dashboard
- GitHub documentation

## Analysis Process
1. Reviewed raw order data and data dictionary.
2. Standardized text fields such as region, channel, status, and category.
3. Converted date and numeric fields into analysis-ready values.
4. Reviewed duplicate orders, missing values, and unusual status records.
5. Built summary analysis for revenue, profit, order count, and average order value.
6. Created a dashboard preview and business insights.

## Key Insights
- Electronics and Furniture make the strongest profit margins.
- Makassar is the strongest region by clean revenue.
- 728 rows required review before being used for dashboard decisions.
- Retail is dominating the sales data compared with other channels.
- Sales in the first quarter of the year are much higher than sales on the second quarter.
